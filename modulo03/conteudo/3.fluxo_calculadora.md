# Projeto Calculadora: Fluxo de Execução

**Turma:** LionsDev  
**Tópicos:** Fluxo de Execução, Menu Interativo, Loops (`while`), `switch`, Acumulador de Resultado.

---

### 1. Visão em Passos (alto nível)

1. Inicializar acumuladores (ex.: resultado em zero).
2. Repetir até o usuário pedir para sair:
   - **2.1)** Mostrar menu de operações no terminal.
   - **2.2)** Ler a opção escolhida (`prompt`).
   - **2.3)** Conforme a opção:
     - Pedir um ou mais números ao usuário;
     - Validar entradas;
     - Chamar a função da operação e atualizar o acumulado;
     - Ou apenas mostrar o resultado;
     - Ou encerrar.
   - **2.4)** Voltar ao menu (continua o loop).
3. Mensagem de encerramento (quando aplicável).

### 2. Diagrama de Fluxo (simplificado)

```
    +------------------+
    |     INÍCIO       |
    +--------+---------+
             |
             v
    +------------------+
    | resultado := 0   |
    +--------+---------+
             |
             v
        +----+----+     op == sair?
        |  Menu   |<----------------+
        +----+----+                 |
             |                      |
             v                      |
        +----+----+                 |
        | Lê opção|                 |
        +----+----+                 |
             |                      |
             v                      |
        +----+----+   não           |
        | switch  +-----------------+
        +----+----+
             |
             v
    +------------------+     inválido
    | Prompt número    +--------+
    +--------+---------+        |
             | válido           |
             v                  v
    +------------------+   +---------+
    | Atualiza resultado|  |  Aviso  |
    +--------+---------+   +----+----+
             |                  |
             +--------+---------+
                      |
                      v
                 (volta ao Menu)
```

### 3. Interatividade, Menu e Múltiplas Operações

- **Interatividade:** a cada volta do loop o usuário vê o menu e decide.
- **Menu:** centraliza as escolhas numéricas (1..N) mapeadas no `switch`.
- **Múltiplas operações:** o `while` mantém o programa vivo; o acumulador guarda o estado entre uma operação e outra.
- **Persistência do resultado:** neste projeto é "em memória" apenas (variável que sobrevive enquanto o processo Node está rodando).

### 4. Onde Está no Código?

`Modulo03/exercicios_slides/calculadora.js`

- Loop principal: `while (op != 7) { ... }`
- Menu: `console.log` com as opções [1]..[7]
- Ramificação: `switch (op)` com cases para cada fluxo
- Opção "ver resultado": case que só exibe resultado sem alterar
